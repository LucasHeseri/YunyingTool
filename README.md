# YunYingTool

运营工具集 — 桌面端图片加工工具。

## 功能

- **WalletKit机模**：将上传的图片合成到卡片/票券 SVG 机模中，支持卡片和票券两种模板切换
- **图片压缩**：等比例压缩图片至指定宽度
- **logo处理**：宫格图（160×160 圆角方形，可选鸿蒙消息提醒红点）和圆形 logo（160×160 圆形裁剪）

## 使用方式

直接在浏览器中打开 `desktop.html` 即可使用，无需安装任何依赖。

## 文件说明

| 文件 | 说明 |
|------|------|
| `desktop.html` | 主页面 |
| `card-original.js` | 卡片 SVG 模板 data URI |
| `ticket-original.js` | 票券 SVG 模板 data URI |
| `Card.svg` | 卡片原始 SVG |
| `Ticket.svg` | 票券原始 SVG |
